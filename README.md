## About

I build full-stack and AI-powered systems, with a particular interest in the intersection of applied AI and security. Most recently that's meant designing a Retrieval-Augmented Generation pipeline for financial document analysis — thinking through chunking strategy, embedding choice, and retrieval quality rather than just wiring an API together.

I'm also spending time in modern C++ outside of coursework: RAII, move semantics, and smart pointers, mostly as a way to understand what's happening underneath the frameworks I use daily.

If you're looking at this because we're talking about an opportunity — the projects below are the clearest signal of how I work. Happy to walk through any of them in more depth.

## Projects

**[FinTrust AI](https://github.com/mayankjoshi18/FinTrust-AI)** — `FastAPI` `React` `ChromaDB` `Gemini`
A full-stack financial document intelligence platform. The core of it is a RAG pipeline built with recursive text chunking, Gemini embeddings, ChromaDB vector search, and MMR reranking, exposed through a JWT-authenticated API with role-based access control.

**[Student Volunteer Management System](https://github.com/mayankjoshi18/Student-Volunteer-Management-System)** — `React` `TypeScript` `Node.js` `MongoDB`
A platform for managing volunteer events end to end — registration, attendance, hour tracking, and PDF certificate generation — with role-based dashboards built in React and TypeScript.

**[File Encryptor](https://github.com/mayankjoshi18/File-Encryptor)** — `C++17` `STL` `std::filesystem`
A command-line tool for recursive file encryption and decryption, used as a vehicle to apply RAII, move semantics, and smart pointers in a real systems-programming context.
